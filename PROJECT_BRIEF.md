# PROJECT_BRIEF.md

## Goal

Build a simple portfolio website for Data Engineer.

The website should help recruiters quickly understand:
- My technical skills
- My projects
- My ability to design pipelines
- My understanding of data quality and reliability
- My communication style

## Positioning

I am focused on building reliable data pipelines and analytics-ready datasets using Python, SQL, orchestration tools, and modern data engineering practices.

## Target Audience

- Technical recruiters
- Hiring managers
- Data engineering teams
- Analytics engineering teams

## Desired Impression

Visitors should think:

“This candidate understands the fundamentals of data engineering and can clearly explain how they build reliable data systems.”

## Site Style

The site should be:
- Minimal
- Technical
- Professional
- Fast
- Easy to navigate

It should not feel like a frontend animation showcase.

## Core Pages

- Home
- Projects
- Case Studies
- About
- Contact / Resume

- View Projects
